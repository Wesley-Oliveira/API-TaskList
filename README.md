# API-TaskList
    Simple API-Rest, created to train skills in nodeJS, using the express framework.
    
 # Routes and Methods
    # Task
        POST, GET - /tasks
        PUT, DEL - /houses/id
    # User
        POST, PUT - /users
    # Session
        POST - /sessions
        
# Install dependencies
    yarn

# Execution after yarn configuration
    For backend execution: yarn dev
